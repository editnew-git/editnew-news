## What You’ll Learn

This guide explains how to add another module to a new or existing layout.

## Before You Begin

Open the layout you want to update, or create a new layout.

## Add a Module

1. Create a new layout or open an existing layout.
2. Select the **Gear** button, then select **Module Manager**.
3. Choose the module you want to add to your layout.
4. Select **Add Module**, then select **Save**.
5. After saving, confirm that the new module appears on the right side of the page.

## Expected Result

The selected module is added to your layout and appears on the right side of the page.
